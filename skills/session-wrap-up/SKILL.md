# Session Wrap-Up

Run this automatically whenever a conversation goes quiet for 10+ minutes or when Curtis says goodbye/thanks/done.

## Steps

1. Update MEMORY.md
 - Add anything new learned this session about Curtis, his projects, preferences
 - Update status of any project or task discussed

2. Update active-tasks.md
 - Mark completed tasks as done with date
 - Update in-progress tasks with current status and next action
 - Add any new tasks mentioned

3. Append to memory/session-log.md
 Format: [YYYY-MM-DD HH:MM AEST] — [topics discussed] — [decisions made] — [pending items]

4. Confirm to Curtis via Telegram:
 "✅ Session saved. Memory updated."
