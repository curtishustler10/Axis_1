# Viral Content Cloning Workflow

## From Alex Djordjevic's Method

### Step 1: Find Viral Content
- Source: TikTok, Instagram Reels, YouTube Shorts
- Criteria: High engagement, relatable, simple concept
- Save to swipe file

### Step 2: Extract Winning Variables
```
Movement patterns (how objects move)
Framing (angle, composition)
Aesthetic (colors, mood, vibe)
Audio type (trending sounds)
```

### Step 3: Generate with AI
- Use: Gemini → extract movement JSON
- Use: Freepik/Nano Banana Pro → generate backgrounds
- Use: Google V3 / Kling 3.0 → video generation

### Step 4: Blend with Product
- Take product photo (match original position)
- Generate background with same aesthetic
- Overlay text + trending audio
